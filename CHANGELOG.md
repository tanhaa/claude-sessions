# Changelog

All notable changes to this project will be documented in this file.

## [0.2.2] - 2026-04-01

### Added
- Textual TUI for browsing and resuming Claude Code sessions
- Session list with search, pin, and hide
- Browse chat history with toggleable side/bottom panel (`v` to toggle, `b` to cycle position)
- ASCII logo inline with search bar
- Config via `~/.config/claude-sessions/config.json`
- Persistent filter state (hidden, pinned, panel position) in `~/.config/claude-sessions/filters.json`
- Config type validation and `os.chdir` error handling
- 56 tests
